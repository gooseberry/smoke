    Download game windows

    1. Open the Chrome browser and go to your [GOG Library].

    2. Select the game you wish to install from your collection to see the download options

    3. Next to the System option, select Windows.

    4. Click on the link under DOWNLOAD OFFLINE BACKUP GAME INSTALLERS.

      Example

    5. Once the download completes, open the Files app and go to Downloads.

    6. Move the downloaded installer to the Linux files folder.
